# Bazy danych

## Prerequisites

- [pnpm](https://pnpm.io/) - package manager
- [Node.js](https://nodejs.org/) - JavaScript runtime

## Getting Started

Install packages:

```bash
pnpm install
```

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
